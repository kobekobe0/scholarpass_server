# Node.js Express MongoDB Socket.IO Server

A simple template for a Node.js server using Express, MongoDB, and Socket.IO.

- Node.js
- Express
- MongoDB
- Socket.IO

│   ├── middleware
│   ├── models
│   ├── routes
│   └── index.js
├── .env
├── .gitignore
├── package.json```
